# DISYS_E02
Mandatory assignment 2 in DISYS.

# The program
The program can run three nodes which are predefined on localhost:8001, localhost:8002 and localhost:8003.

# Run
Open 3 terminal windows in Main/
1. First window type "go run ./Main.go 1"
2. Second window type "go run ./Main.go 2"
3. Third window type "go run ./Main.go 3"

# Send request to do stuff
1. Type "r" in the terminal to get the node to send a request to do stuff.
2. The node will do stuff for seconds.
